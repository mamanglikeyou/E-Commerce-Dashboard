# E-Commerce Data Analysis Dashboard

Dashboard ini dibuat menggunakan **Streamlit** sebagai media penyampaian hasil analisis data e-commerce, yang mencakup segmentasi pelanggan (RFM) dan distribusi revenue berdasarkan wilayah.

## ⚙️ Setup Environment

Sebelum menjalankan dashboard, pastikan Python telah terinstall pada komputer.  
Selanjutnya, install library yang dibutuhkan menggunakan file `requirements.txt`.

Buka terminal atau command prompt pada folder project, lalu jalankan perintah berikut:

pip install -r requirements.txt

## ▶️ Menjalankan Dashboard di Local

Setelah seluruh dependensi berhasil diinstal, jalankan dashboard dengan perintah berikut:

streamlit run dashboard.py

Jika berhasil, browser akan terbuka secara otomatis dan dashboard dapat diakses melalui alamat:

http://localhost:8501
